Flappy Bird Clone
=================


This is a [Flappy Bird] clone I created to learn [Stencyl].
Took around 12 hours and a lot of [Nujabes] to make.

You can check it out at [Gamejolt].


Tools:
------

- [Stencyl]
- [Audacity]
- [as3sfxr]
- [Paint]
